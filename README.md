This python program scrape's the list of films and web series that you are watching in netflix and save it to csv files as well as an excel file and share the 
excel file to your desired gmail account . the program captures cookies when you manually login to netflix and use those cookies until they expire and intimates you
to login manually to capture cookies again . you can enter any profile of your choice and the program then start scraping data like titles , time watched , total
episodes if a series , how many episodes are watched more than half and then saves all this data into csv files and excel file and then send the file to your desired 
gmail .
